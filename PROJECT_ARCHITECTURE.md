# PC Usage Time Logger - Project Architecture

## 📊 プロジェクト概要

Windows Event Log から PC の使用時間を抽出し、日ごとに集計してグラフで可視化するツール

## 🔧 各スクリプトの役割

### 1️⃣ `run_all.py` - メイン実行スクリプト
**推奨実行方法：** `python run_all.py`

```
目的: 全処理の統合・自動実行
機能:
  ✓ pc_usage_logger.py を実行
  ✓ pc_usage_graph.py を実行
  ✓ エラー処理・ステータス表示
  ✓ 順序制御（ログ取得→グラフ生成）
```

---

### 2️⃣ `pc_usage_logger.py` - ログ抽出・集計
**実行方法：** `python pc_usage_logger.py`（管理者権限）

```
目的: Windows Event Log から PC 使用時間を抽出
処理フロー:
  1. PowerShell コマンドで Event Log をクエリ
  2. ロック/解除イベントを時系列でソート
  3. 「解除～ロック」間を計算（使用時間）
  4. 日をまたぐセッションを分割
  5. 日ごとに集計
  6. CSV ファイルに出力

入力:
  - Windows Event Log
    ├─ Microsoft-Windows-Winlogon/Operational (Event 812)
    ├─ Security Log (Event 4800-4803)
    └─ System Log (Event 6006-6008)

出力:
  - pc_usage_log.csv
    日付, 使用時間（時間）, 使用時間（時:分）
```

**注意:** 管理者権限が必須（Event Log 読取用）

---

### 3️⃣ `pc_usage_graph.py` - グラフ生成・表示
**実行方法：** `python pc_usage_graph.py`

```
目的: CSV データをグラフで可視化
処理フロー:
  1. pc_usage_log.csv を読み込み
  2. 直近30日のデータをフィルタ
  3. 2つのグラフを生成:
     ├─ グラフ1: 折れ線グラフ（推移）
     └─ グラフ2: 棒グラフ（平均値との比較）
  4. グラフを PNG で保存
  5. ウィンドウで表示
  6. コンソールに統計情報を出力

入力:
  - pc_usage_log.csv

出力:
  - pc_usage_graph.png（グラフ画像）
  - コンソール出力（統計情報）
```

**グラフの色分け:**
- 紫色（#A23B72）: 平均値以上
- オレンジ色（#F18F01）: 平均値以下
- 赤点線: 平均値ライン

## 📊 データ仕様

### 入力: Event Log イベント
```
Event ID 812 (Winlogon):
  - Type 4: ロック（画面 OFF）
  - Type 5: 解除（画面 ON）
  - Type 6: スクリーンセーバー開始
  - Type 7: スクリーンセーバー終了

Security Log:
  - Event 4800: ロック
  - Event 4801: 解除
  - Event 4802: ロック
  - Event 4803: 解除

System Log:
  - Event 6006: シャットダウン
  - Event 6008: 予期しないシャットダウン
  - Event 42, 506, 109: スリープ終了
```

### 出力: CSV フォーマット
```csv
日付,使用時間（時間）,使用時間（時:分）
2026-06-18,8.50,8:30
2026-06-19,9.25,9:15
2026-06-20,7.75,7:45
```

## 🔄 エラーハンドリング

| エラー | 原因 | 解決策 |
|------|-----|--------|
| `Permission Denied` | 管理者権限なし | 管理者権限で実行 |
| `Event Log が見つかりません` | 該当イベントなし | Event Log 保持期間を確認（通常7日） |
| `CSVファイルが見つかりません` | logger.py 未実行 | logger.py を先に実行 |
| 文字化け | フォント未対応 | `japanize_matplotlib` をインストール |

colors = ['#FF5733' if h >= avg_hours else '#33FF57' for h in hours]

## 🔍 トラブルシューティング

### Q: グラフに日付がすべて表示されない
**A:** `pc_usage_graph.py` の X 軸設定を確認してください
```python
ax.set_xticks(x_pos)
ax.set_xticklabels(date_labels, rotation=0, ha='center', fontsize=12)
```

### Q: Event Log に過去のデータがない
**A:** Windows Event Log の保持期間を確認・延長してください
- イベントビューアー → System → プロパティ → 保持期間

### Q: 「管理者権限が必要」エラー
**A:** PowerShell/cmd を管理者として実行してから実行してください
```powershell
# PowerShell 管理者実行
python pc_usage_logger.py
```
